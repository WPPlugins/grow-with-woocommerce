== Installation ==

Installing "Starter Plugin" can be done either by searching for "Starter Plugin" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

- Download the plugin via WordPress.org.
- Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
- Activate the plugin through the 'Plugins' menu in WordPress
- Grow Settings are found in: WooCommerce > Settings > Integrations for Grow

Google Analytics integration is required and there is a one click integration in the settings. Click the integration button and you will be provided an authorisation code from Google. Copy this and paste into the provided text box.

Once this is added you can then select the correct Google Analytics account you want to link with.

Head to WooCommerce > Grow > Targets to set your targets.
